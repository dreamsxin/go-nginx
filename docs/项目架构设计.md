# 项目架构设计

## 目录结构

```plainText
go-nginx/
├── cmd/
│   └── go-nginx/
│       └── main.go                 # 应用入口点
├── config/
│   ├── parser.go                   # 配置文件解析
│   └── types.go                    # 配置相关数据结构
├── core/
│   ├── server.go                   # 服务器核心逻辑
│   └── listener.go                 # 网络监听器
├── balancer/
│   ├── http/
│   │   ├── balancer.go             # HTTP负载均衡器
│   │   ├── session.go              # HTTP会话保持
│   │   └── healthcheck.go          # HTTP健康检查
│   ├── tcp/
│   │   ├── balancer.go             # TCP负载均衡器
│   │   └── healthcheck.go          # TCP健康检查
│   └── udp/
│       ├── balancer.go             # UDP负载均衡器
│       └── healthcheck.go          # UDP健康检查
├── monitor/
│   ├── metrics.go                  # 性能指标收集
│   └── stats.go                    # 统计数据管理
├── api/
│   ├── handler.go                  # API请求处理器
│   └── router.go                   # API路由
├── util/
│   ├── logger.go                   # 日志工具
│   └── utils.go                    # 通用工具函数
├── test/
│   ├── http_test.go                # HTTP测试
│   ├── tcp_test.go                 # TCP测试
│   └── udp_test.go                 # UDP测试
├── config.yaml                     # 配置文件
├── go.mod                          # Go模块定义
└── README.md                       # 项目文档
```
